# Personal Portfolio

My personal portfolio website — a single-page site showcasing my projects (full-stack and AI/ML), engineering craftsmanship, leadership experience, and contact links.

**Live site:** [a-ariyanto.github.io/portfolio](https://a-ariyanto.github.io/portfolio/)

Built with React 19, TypeScript, Vite, and Tailwind CSS. Deployed to GitHub Pages via GitHub Actions.

## Running locally

From the repository root:

```bash
npm i
npm run dev
```

Then open the local URL Vite prints (default: http://localhost:5173).
